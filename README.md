# 개인회생 예상 탕감률 조회 서비스

개인회생 절차를 고려 중인 분들을 위한 예상 탕감률 조회 서비스입니다.

## ✨ 주요 기능

### 📊 계산 기능
- **라이프니츠식 현가 계산**: 법정이율(연 5%) 적용한 정확한 변제액 계산
- **동적 변제기간**: 36~60개월 자동 조정
- **청산가치 반영**: 자산 가액 기반 최소 변제액 보장
- **최저생계비 적용**: 부양가족 수에 따른 생계비 차감

### 🏠 자산 계산
4가지 주거 형태를 지원합니다:
1. **본인 명의 자가**
   - KB시세 입력
   - 근저당권 반영
   - 실거주 여부 고려

2. **전세**
   - 보증금 입력
   - 지역별 최우선변제금액 차감
   - 임대차보증금 우선변제

3. **월세**
   - 보증금 입력
   - 최우선변제 적용

4. **무상거주**
   - 배우자 명의 주택 여부 확인
   - 법원 관할별 청산가치 차등 적용

### 👨‍👩‍👧‍👦 부양가족 계산
- **결혼 상태 반영**: 기혼/미혼/이혼
- **자녀 수 입력**
- **배우자 소득 여부**
- **법원 관할별 차등 계산**:
  - 주요 법원 (서울/수원/대전/부산): 배우자 소득 無 시 부양가족 +1
  - 기타 법원: 기본 계산식 적용

### 💬 상담 신청
- **카카오톡 채널 연동**
- **자동 메시지 생성**: 입력 정보 자동 요약
- **클립보드 복사**: 원클릭 복사 및 붙여넣기 안내
- **상세 정보 포함**: 모든 입력 정보와 계산 결과 전송

## 🚀 시작하기

### 환경 요구사항
- Node.js 18.x 이상
- npm 또는 yarn

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/hamshmas/hamshams1.git
cd hamshams1

# 의존성 설치
npm install

# 환경 변수 설정
cp .env.example .env.local
# .env.local 파일을 열어 NEXT_PUBLIC_KAKAO_JS_KEY 설정

# 개발 서버 실행
npm run dev
```

개발 서버: http://localhost:3000

### 프로덕션 빌드

```bash
# 빌드
npm run build

# 프로덕션 실행
npm start
```

## 🧮 계산 로직

### 변제액 계산
```typescript
// 라이프니츠식 현가 계산
PV = PMT × [(1 - (1 + r)^-n) / r]

// 변수:
// PV: 현재가치 (총 변제액)
// PMT: 월 변제액 (가용소득)
// r: 월 이율 (연 5% ÷ 12)
// n: 변제 기간 (개월)
```

### 가용소득 계산
```typescript
가용소득 = 월소득 - 최저생계비(부양가족수)
```

### 부양가족 수 계산
```typescript
// 기혼 - 주요법원 - 배우자 소득 無
부양가족수 = 자녀수 + 1

// 기혼 - 기타 경우
부양가족수 = (자녀수 ÷ 2) + 1

// 미혼/이혼
부양가족수 = 자녀수 + 1
```
